PROJECT CONVERSATION

The development of this project began by carrying forward my Midterm practical website into the CAT 2 repository.

Initially, the website only contained HTML and CSS. It displayed company information, products, shipping details, warranty information and contacts.

After setting up the CAT 2 project, JavaScript functionality was added to make the website interactive.

The products were converted into JavaScript objects and displayed dynamically.

A wishlist feature was introduced, allowing users to add products they would like to purchase. Each item added to the wishlist could also be removed.

A contact form was implemented to improve communication between customers and the business. JavaScript validates the form and displays a success message whenever a user enters their name correctly.

An interactive banner was also implemented. Clicking the banner displays or hides a welcome caption.

Throughout development, several challenges were encountered including:

- Incorrect file paths
- GitHub repository setup
- JavaScript linking
- Image loading issues
- Responsive layout adjustments

These challenges were solved through testing, debugging and reorganizing the project structure.

Overall, the project demonstrates practical application of HTML, CSS and JavaScript in building an interactive and responsive business website.
